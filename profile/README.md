# liveeasy

liveeasy is the GitHub organization for collaborative app development across
mobile and web products.

Team workflow:

- Plan work in GitHub Issues.
- Track ownership and status in the liveeasy Project.
- Open Draft PRs early.
- Review overlap before coding.
- Keep durable decisions visible to both collaborators and AI agents.
